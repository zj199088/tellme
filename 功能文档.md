# 健身小程序功能文档

## 项目概述
健身小程序是一款集模板生成、自定义计划、AI生成计划和训练跟踪于一体的综合健身应用。

---

- **前端页面**:
  - [ ] `/pages/login/index.vue` (微信登录页)
  - [x] `/pages/admin/auth.vue` (管理员登录页)
- **后端接口**:
  - [x] `POST /api/auth/wechat` (微信登录)
  - [x] `POST /api/auth/admin` (管理员登录)
